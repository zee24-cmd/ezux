EzLayout

The architectural backbone of your application, providing a high-performance workspace with integrated services.

Service-Driven

Built-in Registry for Theme, Layout, I18n, and Notifications.

Responsive

Intelligent sidebar behavior that adapts from mobile to ultrawide.

IoC Ready

Invert control by injecting your own Header or Sidebar components.

Properties

Name Type Description
componentsEzLayoutComponentsSlots for custom Header, Sidebar, and Footer.
authConfig{ signInSlot, signUpSlot }Configuration for the built-in authentication slider.
headerConfigEzHeaderPropsConfiguration for the default header (logo, search, etc).
contentClassNamestringCustom CSS classes for the main viewport area.

Methods (LayoutService)

Method Arguments Description
toggleSidebar(open?: boolean)Programmatically collapses or expands the side navigation.
setMode(mode: 'dashboard' | 'auth')Switches the global layout context.
setAuthPage(page: 'signin' | 'signup')Toggles the view within the authentication slider.

EzTable

A powerhouse data grid engineered for the most demanding enterprise datasets.

Extreme Scaling

Virtualization allows for 100k+ rows with sub-1ms render times.

Smart Filtering

Excel-style column filters with advanced logical grouping (AND/OR).

Data Integrity

Built-in transaction history with full Undo/Redo support.

Properties

Name Type Description
dataT[]The dataset to display. Can be reactive.
columnsColumnDef[]Highly configurable column definitions with renderers/editors.
paginationbooleanEnables the sticky pagination bar.
enableGroupingbooleanEnables drag-and-drop row grouping.
enableEditingbooleanEnables inline cell editing and transaction tracking.
density'compact' | 'standard'Adjusts row spacing and UI elements.

Events

Event Arguments Description
onDataChange(data: T[])Fired when any row edit is committed.
onRowSelectionChange(rows: Row[])Fired when single or multi-selection states change.
onExport(type: 'csv'|'excel'|'pdf')Fired when the user triggers an export action.

EzScheduler

The ultimate time-management component, combining high-density data with intuitive interaction.

Multi-View

Switch between Day, Week, Month, Timeline, and Agenda views.

Recursion

Native support for complex iCal recursion rules and exceptions.

Fluid Interaction

Pixel-perfect drag-and-drop and resizing with visual ghosts.

Properties

Name Type Description
eventsSchedulerEvent[]The collection of appointment data.
resourcesResource[]Mapping for timeline resource grouping.
currentViewViewThe active view mode (Day, Week, etc).
timeScaleTimeScaleModelConfigures slot intervals and slot count per hour.

Events

Event Arguments Description
actionComplete{ requestType, data }Fired after add, edit, or delete operations.
eventDoubleClick{ data, element }Fired for quick editing of existing events.
navigating{ currentDate, currentView }Fired when the scheduler date range changes.

EzTreeView

Visualize complex hierarchies with full virtualization and deep customization.

Flat-to-Tree

Internally manages flat structures into efficient tree models.

Tri-State Selection

Cascading checkboxes with indeterminate parent states.

Lazy Loading

Fetch nodes on-demand for infinite hierarchical scale.

Properties

Name Type Description
dataTreeNode[]The root-level nodes of your hierarchy.
showCheckboxesbooleanEnables cascading tri-state checkboxes.
allowEditingbooleanEnables inline node renaming on double-click.
searchTermstringBuilt-in search highlighting and filtering.

Methods (via apiRef)

Method Arguments Description
expandAll()Expands every node in the tree.
collapseAll()Collapses the tree back to root levels.
checkNode(id: string, check?: boolean)Imperatively sets the check state of a node.
selectNode(id: string)Sets the primary active selection to a node.
